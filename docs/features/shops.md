# Player Shops

The Player Shops system lets players set up shops where other players can buy items using the server's currency.

---

## Accessing the Shop

Players can browse and interact with shops through the **Community Panel**:

1. Type `!book` to open the Community Panel.
2. Select **Shops**.
3. A list of available player shops will appear.
4. Select a shop to view its inventory and buy items.

---

## Selling Items

Player shops are **buy-only**. Other players can only purchase items from your shop — there is no way for players to sell items into a player shop.

To sell items for money, use the `!sell` command, which works with server shops configured by an admin.

---

## Admin: Managing Player Shops

Administrators can view and manage all player shops from the Admin Panel:

1. Open the **Admin Panel** (`!adminpanel`).
2. Go to **Player Shops Manager**.
3. From here you can:
   - Browse all player shops
   - Review pricing
   - Remove shops if needed

---

## Shop Discounts by Rank

Certain ranks may have a discount multiplier applied to shop transactions. This means VIP or higher-ranked players may pay less or earn more when using shops. This is configured by the server administrator in the **Ranks Editor**.
