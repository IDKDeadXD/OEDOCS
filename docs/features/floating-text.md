# Floating Text (Holograms)

Admins can place floating text entities anywhere in the world. These are sometimes called holograms. They are useful for labeling areas, displaying shop prices, writing rules at spawn, or decorating builds.

---

## Creating Floating Text

Floating text is created and managed through the **Admin Panel**:

1. Open the **Admin Panel** (`!adminpanel`).
2. Go to **Floating Text**.
3. Choose to create a new floating text entry.
4. Enter the text you want to display.
5. The floating text entity will be placed at or near your current location.

---

## Managing Existing Floating Text

From **Admin Panel > Floating Text**, administrators can also:

- View a list of all floating text entities on the server.
- Edit the text of an existing hologram.
- Delete floating text entities that are no longer needed.

---

## Appearance

Floating text appears as a small entity in the world that displays your message above it. The text is visible to all players within a reasonable distance.

---

## Notes

- Only administrators can create, edit, or delete floating text.
- Floating text entities persist in the world until manually removed.
- If a floating text entity is in a location that gets reset or unloaded, it may need to be recreated.
