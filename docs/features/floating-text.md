# Floating Text (Holograms)

Admins can place two types of floating entities anywhere in the world: custom text holograms and live leaderboard displays. Both are managed through the Admin Panel.

---

## Text Holograms

Custom text holograms let you place any message as a floating entity in the world. Useful for labeling areas, displaying rules at spawn, shop signs, or decorating builds.

To create one:

1. Open the **Admin Panel** (`!adminpanel`).
2. Go to **Floating Text**.
3. Choose to create a new text entry.
4. Enter the text you want to display.
5. The hologram will be placed at your current location.

From the same menu you can edit the text or delete existing holograms.

---

## Leaderboard Displays

Leaderboard holograms show a live scoreboard pulled from the server's player stats. The display updates automatically and shows the top players for a chosen stat.

To place a leaderboard:

1. Open the **Admin Panel** (`!adminpanel`).
2. Go to **Floating Text**.
3. Choose to create a leaderboard.
4. Select the stat category to display.
5. The leaderboard will be placed at your current location and update on its own.

---

## Notes

- Only administrators can create, edit, or delete floating text and leaderboards.
- Both types persist in the world until manually removed.
- If an entity is in a chunk that gets reset or unloaded, it may need to be recreated.
