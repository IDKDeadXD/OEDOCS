# Slash Commands

In addition to the `!` prefix chat commands, Obsidian Essentials registers a small set of native Minecraft slash commands accessible directly from the command bar.

---

## Available Slash Commands

### Available to All Players

| Slash Command | Description |
|---|---|
| `/oe:menu` | Opens the Community Panel |
| `/oe:clearchat` | Clears your chat screen |

### Admin Only

| Slash Command | Description |
|---|---|
| `/oe:rank add [Player] [RankName] [Priority]` | Adds a rank to a player |
| `/oe:rank remove [Player] [RankName]` | Removes a rank from a player |
| `/oe:rank list [Player]` | Lists all ranks a player has |
| `/oe:rank priority [Player] [RankName] [Priority]` | Adjusts the priority of a player's rank |
| `/oe:rank create [RankName] [Priority]` | Creates a new rank |

---

## Notes

- Slash commands appear in Minecraft's autocomplete when you type `/oe:`.
- All other features are accessed through `!` chat commands or the in-game panel menus.
- For a full list of chat commands, see [Player Commands](player-commands.md) and [Admin Commands](admin-commands.md).
