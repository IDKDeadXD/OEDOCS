# Server Settings

The Settings menu in the Admin Panel is where you configure server-wide options for Obsidian Essentials.

---

## Accessing Settings

1. Open the **Admin Panel** (`!adminpanel`).
2. Select **Settings**.

---

## Available Settings

### Join Messages

| Setting | What It Controls |
|---|---|
| **Join Message Enabled** | Whether a message is shown to all players when someone joins |
| **Join Message** | The text of the join message. Use `<player>` for the player's name and `<count>` for the current player count |

**Example join message:**
```
<player> has joined! We now have <count> players online.
```

---

### Community Panel

| Setting | What It Controls |
|---|---|
| **Spawn With Book** | Automatically gives players the Community Panel book when they respawn |

---

### Chat

| Setting | What It Controls |
|---|---|
| **Chat Prefix** | The character used before all commands (default: `!`) |
| **Chat Ranks Enabled** | Whether rank tags are shown next to player names in chat |
| **Ranks in Nameplates** | Whether rank tags appear above players' heads |

---

### Economy

| Setting | What It Controls |
|---|---|
| **Money System Name** | The internal name of the currency scoreboard |
| **Kill/Death Money Drop** | Whether money is dropped when a player dies |
| **Death Drop Amount** | How much money is dropped on death |

---

### Vanish

| Setting | What It Controls |
|---|---|
| **Send Leave Message on Vanish** | Whether a fake leave message is sent when an admin vanishes |
| **Server Name in Vanish Messages** | The server/realm name shown in those messages |

---

## Extra Features

The **Extra Features** button in the Admin Panel opens additional configuration options beyond the main settings.

---

## Feature Toggles

The **Feature Toggles** button lets admins enable or disable individual buttons in the Community Panel. Use this to show only the features your server actually uses.

---

## Server Rules

Set your server rules from **Admin Panel > Rules**. Players can view the rules from the Community Panel at any time.

---

## Sidebar

The **Sidebar** option lets admins set up a custom scoreboard sidebar shown to all players. You can put whatever information you want there.
