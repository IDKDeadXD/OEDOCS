# Mob Management

The Mob Settings system lets administrators completely prevent specific mob types from spawning on the server.

---

## Managing Mob Restrictions

1. Open the **Admin Panel** (`!adminpanel`).
2. Go to **Mob Settings**.
3. From here you can:
   - Browse mob categories
   - Enable or disable restrictions for specific mob types

---

## Mob Cash Rewards

Each mob type can be configured to give players money when killed. This is set per mob in the **Mob Settings** menu. When a player kills a mob that has a cash reward configured, the amount is added to their balance automatically.

This works alongside the economy system. Make sure the economy is active on your server for mob cash rewards to function.

---

## Notes

- Mob restrictions apply server-wide.
- Changes take effect immediately after saving.
