# Admin Panel Overview

The **Admin Panel** is the main menu for server administrators. It gives access to every management tool in Obsidian Essentials through an in-game menu system. No external tools needed.

---

## Opening the Admin Panel

Type in chat:
```
!adminpanel
```
or the alias:
```
!ap
```

A book item called **Admin Panel** will appear in your inventory. Hold it and **right-click** (or use it on mobile/console) to open the main admin menu.

> You must have admin permissions or higher to access the Admin Panel.

---

## Admin Panel Menu Options

The Admin Panel contains the following options:

| Button | Description |
|---|---|
| **Public Warps** | Create and manage warp points available to all players |
| **Admin Warps** | Create and manage admin-only warp points |
| **Chat Announcements** | Send formatted announcements to all players |
| **Ban** | Ban and unban players |
| **Players** | View and manage individual players |
| **Lag Zapper** | Remove dropped items and entities to improve performance |
| **Reports** | View player-submitted reports |
| **Logs** | View a history of admin actions |
| **Land Claims** | View and manage all player land claims |
| **Floating Text** | Place and manage floating text (holograms) in the world |
| **Spawn Protection** | Configure the protected zone around spawn |
| **Mob Settings** | Restrict or ban specific mob types |
| **Item Banning** | Configure items that are automatically removed from inventories |
| **Player Shops Manager** | View and manage all player shops |
| **Custom Commands** | Create custom chat commands |
| **Kits Config** | Create, configure, and delete item kits |
| **Ranks Editor** | Create and configure ranks and permissions |
| **Polls** | Create polls for players to vote on |
| **Feature Toggles** | Enable or disable individual Community Panel buttons |
| **Rules** | Set and display server rules |
| **AFK Settings** | Configure the AFK detection and kick system |
| **Sidebar** | Create a custom scoreboard sidebar |
| **Extra Features** | Additional server configuration options |
| **Settings** | General server settings |

---

## Managing Individual Players

1. Open the **Admin Panel**.
2. Go to **Players**.
3. Select the player from the list.
4. A player management menu will appear with options to view their inventory, rank, warnings, and balance.

---

## Admin Mode Toggle

Administrators can switch between admin mode and normal play:

```
!admintoggle
```
or:
```
!at
```

This saves your survival inventory and switches you to creative mode. Running the command again restores your inventory and returns you to survival.

---

## Community Panel (Player Menu)

The Community Panel is the standard player-facing menu. Players access it with `!book`. It contains the following options:

| Button | Description |
|---|---|
| **Polls** | View and vote on server polls |
| **Warps** | Access public warp points |
| **Homes** | Manage personal home locations |
| **Land Claims** | View and manage your land claims |
| **TPA** | Send and manage teleport requests |
| **Shops** | Browse and use player shops |
| **Reports** | Submit a report about another player |
| **Rules** | View server rules |
| **Stats** | View your personal statistics |
| **Credits/Info** | View add-on information |
| **Settings** | Personal UI preferences |
| **Admin Panel** | Shortcut to the Admin Panel (admins only) |

Admins can enable or disable individual Community Panel buttons from **Admin Panel > Feature Toggles**.
