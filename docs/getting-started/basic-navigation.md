# Basic Navigation

This page explains how to navigate the two main menus in Obsidian Essentials and how to use commands.

---

## The Two Main Menus

Obsidian Essentials is controlled through two book items that open interactive in-game menus.

### Community Panel

The **Community Panel** is the player-facing menu. It gives players access to:

- Homes
- Teleport requests
- Public warps
- Player shops
- Personal stats
- Reports
- Personal settings

**How to open it:**

Type in chat:
```
!book
```

A book item will be added to your inventory. Hold it and **right-click** (or use it on mobile/console) to open the menu.

---

### Admin Panel

The **Admin Panel** is the administrator menu. It gives admins access to server management tools including bans, rank editing, settings, custom commands, and more.

**How to open it:**

Type in chat:
```
!adminpanel
```
or the alias:
```
!ap
```

Hold the book and **right-click** (or use it) to open the menu.

> Only players with admin permissions or higher can access the Admin Panel.

---

## Using Commands

All text-based commands use a **prefix** character typed before the command name. The default prefix is `!`.

**Example commands:**
```
!help
!balance
!home spawn
!tpa PlayerName
```

Commands are typed directly into the chat box and sent like a normal message.

---

## Getting a Command List

### Player Commands

Type in chat:
```
!help
```

This shows all commands available to your current rank and permission level.

### Admin Commands

If you are an admin, type:
```
!adminhelp
```
or the alias:
```
!adc
```

---

## Slash Commands

Many features also support native Minecraft slash commands using the `/oe:` prefix. For example:

```
/oe:help
/oe:balance
/oe:tpa PlayerName
```

These work the same as their `!` equivalents. See the full [Slash Commands](../commands/slash-commands.md) reference for a complete list.

---

## Navigating Menus

All menus in Obsidian Essentials use Minecraft's built-in form UI system:

- **Buttons** - Click or tap a button to open a submenu or trigger an action.
- **Text fields** - Type into text boxes to enter names, amounts, or messages.
- **Toggle switches** - Click to turn features on or off.
- **Back buttons** - Return to the previous menu.
- **Close / Cancel** - Exit the current menu without making changes.

No mouse cursor is needed on console or mobile. Use your normal interaction controls.

## Quick Reference

| What you want to do | What to type |
|---|---|
| Open the Community Panel | `!book` |
| Open the Admin Panel | `!adminpanel` or `!ap` |
| See all player commands | `!help` |
| See all admin commands | `!adminhelp` |
| Check your balance | `!balance` |
| Teleport to a home | `!home [name]` |
| Mark yourself as AFK | `!away` |
